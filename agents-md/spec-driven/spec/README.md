# 스펙 인덱스

이 문서는 프로젝트의 모든 스펙 문서에 대한 인덱스입니다.

## 디렉토리 구조

| 디렉토리 | 설명 | 주요 문서 |
|----------|------|----------|
| `overview/` | 아키텍처, 개념, 기술스택 | - |
| `contracts/` | 데이터 계약, 인터페이스 | - |
| `operations/` | 설정, 관측성, 마일스톤 | - |
| `tasks/` | 작업 단위 스펙 | - |
| `decisions/` | ADR | - |
