# 히스토리 아카이브 에이전트

30일 이상 된 세션 로그를 월별 요약으로 압축하고 아카이브합니다.

## 실행 절차

1. `agent/history/sessions/` 내 30일 이상 된 파일을 찾습니다.
2. 월별 요약을 생성합니다: `agent/history/archive/YYYY-MM_summary.md`
3. 원본 세션 파일을 삭제합니다.

## 아카이브 형식

```markdown
# YYYY-MM 월간 요약

## 주요 성과
- [성과 1]
- [성과 2]

## 주요 결정
- [결정 1]
- [결정 2]

## 통계
- 총 세션: N회
- 주요 변경 파일: [파일 목록]

## 세션 목록
| 날짜 | 요약 |
|------|------|
| MM-DD | (1줄 요약) |
```

## 안전 규칙

- **30일 미만 파일은 절대 건드리지 않습니다**
- 아카이브 전 요약을 먼저 생성합니다
- 요약 생성 실패 시 원본을 삭제하지 않습니다
