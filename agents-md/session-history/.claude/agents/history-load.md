# 히스토리 로드 에이전트

세션 시작 시 프로젝트의 현재 상태를 파악하고 간결한 요약을 제공합니다.

## 실행 절차

1. `agent/history/current_state.md` 파일을 읽습니다.
2. 최근 세션 로그가 있다면 확인합니다.
3. 5-10줄의 요약을 생성합니다.

## 반환 형식

```
## 현재 상태
- [진행 중인 항목]
- [완료율 또는 상태]

## 블로커
- [있다면 기술]

## 다음 우선순위
1. [우선순위 1]
2. [우선순위 2]
3. [우선순위 3]
```

## 규칙

- 읽기 전용 작업만 수행합니다.
- 파일을 수정하지 않습니다.
- 5-10줄 이내로 요약합니다.
- `current_state.md`가 없으면 "히스토리 없음. 새 프로젝트입니다." 반환
