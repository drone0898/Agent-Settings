# <프로젝트명> - Web Frontend

Prefer retrieval-led reasoning over pre-training-led reasoning.

## 문서 인덱스
| 경로 | 용도 |
|------|------|
| src/components/ | UI 컴포넌트 |
| src/pages/ | 페이지 컴포넌트 |
| src/hooks/ | 커스텀 훅 |
| src/stores/ | 상태 관리 |
| src/services/ | API 서비스 |
| src/types/ | 타입 정의 |
| tests/ | 테스트 |

## 기술 스택
Framework: <React/Vue/Angular> | TypeScript | <Tailwind/CSS Module> | <Vite/Webpack>

## 핵심 규칙
- 함수형 컴포넌트, 타입 정의 필수
- 컴포넌트: PascalCase, 훅: use 접두사
- 테스트: `*.test.ts(x)`, E2E: `*.e2e.ts`

## 명령어
| 명령 | 용도 |
|------|------|
| `npm run dev` | 개발 서버 |
| `npm run build` | 빌드 |
| `npm run test` | 테스트 |
| `npm run lint` | 린트 |
