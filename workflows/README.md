# 워크플로우 패턴

완성형 워크플로우 패턴 모음입니다.

## 워크플로우 목록

| 워크플로우 | 설명 |
|-----------|------|
| [automated-pr-loop](./automated-pr-loop/) | 자동화된 PR 리뷰 및 수정 루프 |
| [tdd-loop](./tdd-loop/) | 테스트 주도 개발 루프 |

## 워크플로우 vs 커맨드

- **워크플로우**: 여러 단계의 복합 작업 (예: PR 생성 → 리뷰 → 수정 → 재리뷰)
- **커맨드**: 단일 작업 (예: `/commit`, `/test`)

## 사용법

1. 워크플로우 디렉토리의 설정 파일들을 프로젝트에 복사
2. 필요한 플러그인 설치
3. 커맨드 파일을 통해 워크플로우 실행
