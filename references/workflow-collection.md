# 57개 프로덕션 레디 멀티에이전트 워크플로우

> **출처**: [wshobson/commands](https://github.com/wshobson/commands)

## 개요

15개 카테고리에 걸친 57개의 프로덕션 레디 워크플로우 컬렉션입니다.
7+ 전문 에이전트가 협업하여 복잡한 개발 시나리오를 자동화합니다.

## 주요 워크플로우

| 워크플로우 | 설명 |
|-----------|------|
| `feature-development` | 요구사항 분석부터 배포까지 전체 기능 개발 |
| `full-review` | 코드, 테스트, 보안, 성능 종합 리뷰 |
| `tdd-cycle` | 테스트 주도 개발 자동화 사이클 |
| `performance-optimization` | 성능 프로파일링 및 최적화 |
| `incident-response` | 장애 대응 및 분석 |

## 에이전트 역할

- **요구사항 분석가**: 요구사항 명세화
- **아키텍트**: 설계 및 구조 결정
- **구현자**: 코드 작성
- **테스터**: 테스트 작성 및 검증
- **보안 분석가**: 보안 취약점 점검
- **성능 분석가**: 성능 최적화
- **배포 관리자**: 배포 자동화

## 활용 방법

```bash
/workflows:feature-development
/workflows:full-review
/workflows:tdd-cycle
```
