# 실시간 사용량 Statusline

> **출처**: [fcakyon/claude-codex-settings](https://github.com/fcakyon/claude-codex-settings)

## 개요

Claude Code CLI에서 실시간으로 API 사용량을 모니터링하는 statusline 설정입니다.

## 표시 정보

- 계정 전체 블록 사용량
- 세션 컨텍스트 사용률 (%)
- 현재 세션 비용
- 사용량 리셋까지 남은 시간

## 활용

- API 제한 도달 전 작업 복잡도 조절
- 5시간 사용량 블록 리셋 타이밍 파악
- 비용 효율적인 워크플로우 관리
