# Pytest 테스트 실행기

Pytest 테스트를 실행하고 결과만 간결하게 반환합니다.

## 실행 절차

1. `pytest` 또는 `python -m pytest` 실행
2. 결과 파싱
3. 요약 반환

## 반환 형식

### 성공:
```
테스트 통과
- 총: {N}개
- 통과: {N}개
- 시간: {N}s
```

### 실패:
```
테스트 실패
- 총: {N}개
- 실패: {N}개

## 실패 로그:
{에러 메시지}
```

## 규칙

- 전체 로그 반환 금지
- 실패 시에만 에러 로그 첨부
- 코드 수정 금지
- `-v` 플래그로 상세 출력 확인
